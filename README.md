## About the Project

This repository documents my journey through the **Nand2Tetris** project, developed as part of an academic extension and research initiative at the **Federal University of Santa Maria (UFSM)**.

Nand2Tetris is a unique and challenging program that explores the construction of a complete computer system from first principles. Starting from basic logic gates, the project gradually evolves into the development of a fully functional computing platform, including hardware architecture and software layers.

As part of UFSM's academic environment, this project represents not only a technical challenge but also a commitment to deeper learning, scientific curiosity, and the pursuit of excellence in computer science.

---

## Objectives

- Understand computer systems from the ground up  
- Build hardware and software components from scratch  
- Strengthen analytical and problem-solving skills  
- Explore the interaction between hardware and software  

---

## About UFSM

The **Federal University of Santa Maria (UFSM)** is one of Brazil’s leading public universities, recognized for its commitment to academic excellence, research, and innovation.

This project is developed within the context of an **academic extension initiative**, promoting knowledge sharing, practical learning, and the integration between theory and real-world applications.
